API

Mallan.js

mallanJS核心库

在window下注册命名空间Mallan和$,两者等价,提供部分常用工具函数，以及mallanJS核心模块
$只是一个快捷方式，任何用到$的地方都有其可取代的标准方式

Function Mallan(selector [,context])

@param：selector;[必须];类型：String;自定义类型的名称或者选择器表达式

@param：context;[可选];类型：String;自定义类型的名称或者选择器表达式

@returns:自定义对象或者element对象

Mallan函数先尝试选择自定义对象(extendCustom),如果没有对应的自定义对象，则使用选择器，返回element对象

示例： 
//选择自定义对象，返回Mallan.util.cookie对象
$("cookie").get('username');
//选择器，返回element对象
$("#code").addClass("someclass");

Function Mallan.extend(obj)

@param：obj;[必须];类型：Object;

@returns:null

Mallan.extend函数扩展Mallan对象，为其添加属性

示例： 
//为Mallan添加require方法
$.extend({
    "require":function(){
    //something to do
    }
});

Function Mallan.nameSpace.pack(location,obj)

@param：location;[必须];类型：String; 必选以Mallan.开头,否则会抛出异常

@param：obj;[必须];类型：Object;

@returns:null

Mallan.nameSpace.pack函数复制将obj挂载到指定的命名空间下

示例： 
//将cookie对象挂载到Mallan.util.cookie下，如果本地不存在util，则会先创建Mallan.util对象，
 再为Mallan.util.cookie赋值为cookie
$.nameSpace.pack("Mallan.util.cookie",cookie);

Function Mallan.extendCustom(obj)

@param：obj;[必须];类型：Obejct; Object必选有3个属性：name,cls,constructType

@returns:null

Mallan.extendCustom扩展自定义对象

示例： 
//为Mallan扩展自定义对象cookie
$.extendCustom({
    name: "cookie",
    cls: Cookie,
    constructType: "single"
});
//这样就可以方便的使用$()方法来使用cookie
$("cookie").get(key);
$("cookie").set(key,value);

Function Mallan.singleton(init)

@param：init;[必须];类型：Function; 构建单例对象的初始化函数

@returns:Object

Mallan.singleton函数创建一个singleton对象，该对象有一个getInstance方法,可结合Mallan.extendCustom一起使用

示例： 
//创建一个单例cookie对象
var cookie = $.singleton(function(){
//TODO
    return {
        get:function(){},
        set:function(){}
    }
});
//使用cookie
var cookieInstance = cookie.getInstance();
cookieInstance.get(key);
//进一步将其扩展到自定义对象
$.extendCustom({
    name:'cookie',
    cls:cookie,
    constructType:'single'
});
//可以这样使用cookie。Mallan函数将自动执行cookie.getInstance()
$('cookie').get(key);

Function Mallan.getScript(url [,callback])

@param：url;[必须];类型：String; 脚本地址

@param：callback;[可选];类型：Function; 脚本加载后的回调函数

@returns:null

Mallan.getScript函数动态异步加载一个js脚本

示例： 
//加载一个脚本
$.getScript('someurl',function(){
//TODO
});

Function Mallan.require(modules [,callback])

@param：modules;[必须];类型：String;模块的名称，多个模块用逗号隔开

@param：callback;[可选];类型：Function;

@returns:null

Mallan.require 核心模块，动态加载所需的模块及其依赖模块。如果该模块已经加载则直接callback，如果该模块未加载，则加载相应脚本后callback。 任意模块不会加载2次。同时只会有一个异步require正在请求。请求过程中的其他require请求会自动将callback函数push到回调队列或者合并过程中的所有require，等此次require完成后一起发出。 使用require可以使你不必关心模块之间的依赖关系，也不必关心模块是否加载过。

示例： 
//使用photoSlide模块
$.require('plugin.photoSlide',function(){
//TODO
    new $.plugin.photoSlide(dom,options);
});

animate.js

mallanJS 动画库

提供创建动画的类库，并扩展Mallan.dom.element对象，提供方便的调用方式

Class Mallan.animation.animate(dom)

@param：dom;[必须];类型：HTMLDOMElement;

使用new Mallan.animation.animate(dom)来实例化一个animate对象，然后可以调用对象的方法来实现动画效果

示例： 
//实例化对象
var ani = new Mallan.animation.animate(oneDiv);
//添加动画完成的回调函数
ani.events.onComplete.bindOnce(function(){
    console.log('完成');
});

Function Mallan.animation.animate.start(keys,values,time)

@param：keys;[必须];类型：String or Array; css属性，可以是单个string,也可以是数组

@param：values;[必须];类型：String or Array; css属性，可以是单个string,也可以是数组;数量必须与keys保持一致

@param：time;[必须];类型：Number 多少时间内完成动画，单位ms

使用animate对象的start方法来实现动画效果

示例： 
//实例化对象
var ani = new Mallan.animation.animate(oneDiv);
//添加动画完成的回调函数
ani.events.onComplete.bindOnce(function(){
    console.log('完成');
});
//开始动画，使div的宽高在300ms内增长至300px
ani.start(['width','height'],['300px','300px'],300);

Function Mallan.animation.animate.stop()

使用animate对象的stop方法来停止动画

示例： 
//实例化对象
var ani = new Mallan.animation.animate(oneDiv);
//添加动画完成的回调函数
ani.events.onComplete.bindOnce(function(){
    console.log('完成');
});
//开始动画，使div的宽高在300ms内增长至300px
ani.start(['width','height'],['300px','300px'],300);
//停止动画
ani.stop();

Function Mallan.dom.element.animate(keys,values,time,onComplete)

@param：keys 同animate.start方法

@param：values 同animate.start方法

@param：time 同animate.start方法

@param：onComplete [可选];类型：Function,完成所有动画后的回调函数

扩展Mallan.dom.element对象，为其提供animate方法以实现方便的调用

示例： 
//使所有div的宽高在300ms内增长至300px
$("div").animate(['width','height'],['300px','300px'],300,function(){
    console.log('all complete');
})

element.js

element对象，封装HTMLDOMElement，并提供丰富的API和方便的扩展性

Class Mallan.dom.element(selectStr,context)

@param：selectStr;[必须];类型：String or HTMLDOMElement or element or HTMLDOMElement Array or window;

@param：context;[可选];类型：HTMLDOMElement;Dom上下文，默认为document,仅在第一个参数为string时有效

创建element对象实例，对HTMLDOMElement进行封装

示例： 
//选择div下的someclass类下的a元素，并封装为element对象
var el = new Mallan.dom.element(Mallan.dom.selector('div .someclasee a'));
//也可以用$快捷方式
var el = $("div .someclasee a");
//使用el[index]可访问element对象中的HTMLDOMElement元素，el.length返回元素个数
var firstElement = el[0];
var totalNumber = el.length

Function Mallan.dom.element.join(el)

@param：el;[必须];类型：HTMLDOMElement or NodeList or HTMLDOMElement Array or window;

@returns：this

为element对象添加dom元素

示例： 
//选择div下的someclass类下的a元素，并封装为element对象
var el = new Mallan.dom.element(Mallan.dom.selector('div .someclasee a'));
//添加一个元素
el.join(someDom);

Function Mallan.dom.element.each(fn)

@param：fn;[必须];类型：Function;函数引用

@returns：this

循环element对象中的元素，执行fn方法，fn方法中this指向当前元素，fn可接收参数i，在fn中return false可以终止循环

示例： 
//选择div下的someclass类下的a元素，并封装为element对象
var el = new Mallan.dom.element(Mallan.dom.selector('div .someclasee a'));
//调用each方法
el.each(function(i){
    console.log('这是第'+(i+1)+'个元素');
    console.log(this.innerHTML);
});

@param：el;[必须];类型：HTMLDOMElement;

@returns：Number元素在element中的序号

返回el在element对象中的序号，如果不存在则返回-1

示例：

Function Mallan.dom.element.query(selectStr)

@param：selectStr;[必须];类型：String;选择器表达式

@returns：Mallan.dom.element新的element对象

在element对象的第一个元素中查找表达式中的元素，构成新的element对象

Function Mallan.dom.element.exist(fn)

@param：fn;[必须];类型：Function;

如果element对象中有元素，则执行fn，fn中的this指定当前element对象。此方法可以避免繁琐的判断dom是否存在的代码

示例： 
//选择id为test的元素，如果存在则进行一些操作
$("#test").exist(function(){
    this.join(anotherDom);
})

Function Mallan.dom.element.extend(obj)

@param：obj;[必须];类型：Object;

此方法可以扩展element的prototype，Mallan.dom.element下的style,node都是使用此方法扩展的

attribute.js

扩展element对象，提供dom属性相关的api

Function Mallan.dom.element.attr(key,value)

@param：key;[必须];类型：String or Object;

@param：value;[可选];类型：String;

提供Attribute的get和set 操作，若key是对象，则为element中的每一个元素设置key中的属性，若key为String且value不存在，则返回第一个元素的attribute，若key，value都为String，则设置每个元素的attribute

示例： 
//假设el为element对象，设置属性mykey为myvalue
el.attr('mykey','myvalue');
//也可以这样设置多个属性
el.attr({
    'key1':'value1',
    'key2':'value2'
});
//获取第一个元素的mykey属性
var myvalue = el.attr('mykey');

Function Mallan.dom.element.val(value)

@param：value;[可选];类型：String;

提供value的get和set 操作，若有参数则设置每个元素的value为value，若无参数，则返回第一个元素的value

示例： 
//假设el为element对象，设置value为myvalue
el.val('myvalue');
//获取第一个元素的value
var myvalue = el.val();

create.js

扩展element对象，提供用html代码或者tag创建元素

Function Mallan.dom.element.create(html)

@param：html;[必须];类型：String;

@returns：Mallan.dom.element对象

创建元素的便捷api，若参数为html代码，则创建相应的html元素，返回的element对象中为html代码中的顶级元素，若参数为元素tag，则创建相应元素，封装成element对象

示例： 
//创建2个a元素，并未其绑定click事件
var el = Mallan.dom.element.create('<a class="link1">第一个链接</a><a class="link2">第二个链接</a>');
//插入到body中并绑定click事件
el.appendTo(document.body).bind('click',function(e){
  alert(this.innerHTML);
});

full.js

使元素的大小撑满整个页面

Function Mallan.dom.element.full(changeOnResize)

@param：changeOnResize;[可选];类型：Boolean;默认为true

通常用于遮罩，覆盖整个页面大小，changeOnResize为true则当窗口大小变化时自动调整元素大小

示例：

var el = $.dom.element.create("div");
